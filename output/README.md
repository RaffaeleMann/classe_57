#Italian Sentence Generation – Verb Class 57

This directory contains the outputs generated by the evaluated language models for the Italian sentence generation based on **Verb Class 57**.

## Benchmark Description

The benchmark evaluates the ability of language models to generate grammatically correct and semantically plausible Italian sentences while respecting a given target verb and verbal tense: Presente Indicativo, Passato Prossimo, Trapassato prossimo, Trapassato Remoto e Futuro Anteriore and respectively equivalent to Present Simple, Present Perfect, Past Perfect, Past Anterior and Future Perfect. 

The test set is based on **Verb Class 57**, which contains **50 verbal usages** 

Verb Class 57 includes verbs traditionally described as **motion verbs** or **locative verbs**, where the infinitival complement expresses the destination or purpose of the movement.
